Terraform: Infrastructure as Code

What Does Terraform Do?

  • Defines entire AWS infrastructure in code
  • Creates VPC, subnets, gateways, ALB, ASG, EC2
  • Manages security groups and IAM roles
  • Generates Ansible inventory from instances
  • Orchestrates Ansible deployment
  • Validates health checks

Key Files

main.tf ‚Üí Core infrastructure resources variables.tf ‚Üí Configuration variables outputs.tf ‚Üí Exported values deployment.tf ‚Üí Deployment orchestration terraform.tfvars ‚Üí Your configuration values

Traefik: Reverse Proxy & Load Balancer

What is Traefik?

A modern HTTP reverse proxy and load balancer that:

  • Routes incoming traffic to correct services
  • Handles SSL/TLS encryption (HTTPS)
  • Automatically manages certificates
  • Provides load balancing
  • Offers middleware (rate limiting, CORS, etc.)

Flow

User Request (HTTPS) ‚Üì Traefik (Port 443) ‚Üì Check routing rules ‚Üì Forward to correct service (frontend/api) ‚Üì Service returns response ‚Üì Traefik sends back to user

SSL/TLS Certificates

Security & HTTPS

  • SSL/TLS: Encrypts communication between user and server
  • HTTPS: HTTP over SSL/TLS
  • Certificates: Verify server identity
  • Let's Encrypt: Free, automatic certificate renewal

In This Project

‚Ä¢ Traefik manages SSL certificates ‚Ä¢ Let's Encrypt provides free certificates ‚Ä¢ Automatic renewal before expiration ‚Ä¢ HTTP redirects to HTTPS ‚Ä¢ All traffic encrypted

Result: Users see a secure lock üîí in their browser.

How to Deploy: Step by Step

Prerequisites (5 minutes)

# Install tools brew install terraform awscli ansible jq # Configure AWS export AWS_ACCESS_KEY_ID="your-key" export AWS_SECRET_ACCESS_KEY="your-secret" export AWS_REGION="us-east-1"

Initialize (5 minutes)

cd infra make setup-backend terraform init terraform validate

Deploy (10-30 minutes)

terraform apply -auto-approve

Configuration: terraform.tfvars

# Infrastructure aws_region = "us-east-1" project_name = "devops-stage-6" environment = "dev" instance_type = "t3.medium" asg_desired_capacity = 2 # Deployment instance_ready_timeout = 300 enable_ssl = true ssl_provider = "letsencrypt" traefik_acme_email = "your-email@example.com" # Validation deployment_health_check_retries = 30 deployment_log_level = "info"

Note: All variables have sensible defaults. Customize as needed.

Disaster Recovery & Rebuilding

Complete Disaster: Everything Down

# Step 1: Destroy everything terraform destroy -auto-approve # Step 2: Rebuild from scratch terraform apply -auto-approve # Result: Brand new infrastructure in 10-30 minutes

Partial Recovery: Instance Down

  • Auto Scaling Group detects failure
  • Automatically launches replacement instance
  • Ansible deploys to new instance
  • Service restored automatically
